Installation

Prerequisites

Make sure Node.js is installed:

node --version


Clone the Repository

git clone https://github.com/hasty-joel/Club_nitch.git
cd Club_nitch
npm install


Configuration

If environment variables are required:

cp .env.example .env.local

Add necessary API keys or configuration values inside the .env.local file.


Run Locally

npm run dev

Open your browser and visit:
http://localhost:3000


Build for Production

npm run build

Preview production build:
npm run preview


Deployment

This project can be deployed on platforms such as Vercel, Netlify, or any static hosting provider.
